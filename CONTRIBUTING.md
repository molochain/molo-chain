# Contributing to MOLO CHAIN

Thank you for your interest in contributing to MOLO CHAIN!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add your feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## Code Standards

- Follow Solidity best practices
- Use OpenZeppelin contracts where possible
- Write tests for all new functionality
- Document public functions with NatSpec

## Security

If you discover a security vulnerability, please email security@molochain.com instead of opening a public issue.
